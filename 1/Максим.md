# Характеристики

| Название | Показатель          | Модификатор              |
| -------- | ------------------- | ------------------------ |
| ИНТ      | `INPUT[number:ИНТ]` | `VIEW[({ИНТ} - 10) / 2]` |
| СИЛ      | `INPUT[number:Сил]` | `VIEW[({Сил} - 10) / 2]` |
| ЛВК      | `INPUT[number:ЛВК]` | `VIEW[({ЛВК} - 10) / 2]` |
| ХАР      | `INPUT[number:ХАР]` | `VIEW[({ХАР} - 10) / 2]` |

| `INPUT[number:HP]` | `INPUT[number:max_HP]` |

# Роль и Навыки

## Роль
`INPUT[text:Роль]`

## Способность
`INPUT[textArea:Способность]`
## Навыки

| Навык           | Бонус характеристик      | Когда кидать                     | Мастерство      |
| :-------------- | :----------------------- | :------------------------------- | --------------- |
| **Атлетика**    | `VIEW[({Сил} - 10) / 2]` | Лазаешь, прыгаешь, ломаешь двери | `INPUT[number]` |
| **Бой**         | `VIEW[({Сил} - 10) / 2]` | Атаки ближнего боя               | `INPUT[number]` |
| **Скрытность**  | `VIEW[({ЛВК} - 10) / 2]` | Крадешься, прячешься             | `INPUT[number]` |
| **Стрельба**    | `VIEW[({ЛВК} - 10) / 2]` | Атаки дальнего боя               | `INPUT[number]` |
| **Техника**     | `VIEW[({ИНТ} - 10) / 2]` | Чинишь, разбираешься в механике  | `INPUT[number]` |
| **Медицина**    | `VIEW[({ИНТ} - 10) / 2]` | Лечишь, стабилизируешь           | `INPUT[number]` |
| **Восприятие**  | `VIEW[({ИНТ} - 10) / 2]` | Ищешь засаду, замечаешь детали   | `INPUT[number]` |
| **Убеждение**   | `VIEW[({ХАР} - 10) / 2]` | Врешь, уговариваешь              | `INPUT[number]` |
| **Запугивание** | `VIEW[({ХАР} - 10) / 2]` | Пугаешь, давишь                  | `INPUT[number]` |
