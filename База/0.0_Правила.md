# БАЗОВЫЕ ПРАВИЛА

Этот файл — всё, что нужно знать для игры. Остальное (роли, снаряжение, импланты) лежит в соседних файлах.

---

## 1. ГЛАВНОЕ ПРАВИЛО

Если ты хочешь сделать что-то, кидай **1d20** и добавляй модификатор нужной [[0.0_Правила#2. ХАРАКТЕРИСТИКИ|Характеристики]]. Результат сравнивай со сложностью, которую загадывает Мастер.

| Сложность | Когда бывает |
| :--- | :--- |
| **10** | Легко |
| **12** | Средне |
| **15** | Сложно |
| **18** | Опасно |
| **20+** | Легендарно |

---

## 2. ХАРАКТЕРИСТИКИ

У тебя есть 4 характеристики. Значение от 0 до 20. 
**Модификатор** считается как (Значение - 10)/2 (окр. вниз).

| Характеристика | На что влияет |
| :--- | :--- |
| **СИЛ** (Сила) | Ближний бой, здоровье, взлом дверей, переноска тяжестей |
| **ЛОВ** (Ловкость) | Стрельба, инициатива, скрытность, метание гранат |
| **ИНТ** (Интеллект) | Хакинг, ремонт, внимательность, управление дронами |
| **ХАР** (Харизма) | Разговоры, запугивание, торговля, запугивание |

---

## 3. ЗДОРОВЬЕ, БРОНЯ И ИНВЕНТАРЬ

### Хиты и броня

- **Хиты (ХП/HP):** `20 + (Модификатор СИЛ × 2)`. Если HP упали до 0, ты без сознания.
- **Класс Брони (КБ/AC):** `10 + Модификатор ЛОВ + Броня`. Чем выше КБ, тем сложнее в тебя попасть.
- **Броня бывает:**
    - Легкая: +2 к КБ
    - Средняя: +3 к КБ
    - Тяжелая: +4 к КБ

Подробнее — в [[0.1_Снаряжение#1. БРОНЯ|Снаряжении]].

### Инвентарь

У тебя есть ограниченное место для снаряжения. Все предметы занимают слоты.

| Тип | Слотов |
| :--- | :--- |
| Без сумки | 6 слотов |
| С сумкой или обычной разгрузкой | 10 слотов |
| С [[01_Снаряжение#1. БРОНЯ|Тактической разгрузкой]] | 12 слотов |

Оружие в руках не занимает слоты. Броня и импланты не занимают слоты — они всегда на тебе.

Сколько слотов занимает каждый предмет — смотри в [[0.1_Снаряжение#10. ИНВЕНТАРЬ|таблице инвентаря]].

## 4. БОЙ

### Как ходить
В твой ход можно сделать **ОДНО действие** и **ПЕРЕДВИНУТЬСЯ** до 9 метров.

**Действия:**
- Атаковать
- Использовать предмет
- Активировать имплант
- Взломать
- Перезарядить оружие
- Отдать команду дрону

### Как атаковать
1. Кидаешь **1d20 + Модификатор ЛОВ** (для стрельбы) или **+ СИЛ** (для ближнего боя).
2. Если результат **равен или выше КБ врага** — попал.
3. Кидай урон оружия (кубик указан в файле снаряжения).

**Критический выстрел:** если на кубе выпало **20**, урон удваивается.

### Перезарядка
У каждого оружия есть емкость магазина:

| Оружие | Магазин |
| :--- | :--- |
| Пистолеты | 12 патронов |
| Пистолеты-пулеметы | 30 патронов |
| Дробовики | 6 патронов |
| Винтовки | 25 патронов |

Перезарядка — действие. Если у тебя есть [[0.2_Импланты#Личный порт|Личный порт]], можно перезарядить подключенное оружие бонус-действием.

### Гранаты и взрывы
Метание гранаты — атака дальнего боя (ЛОВ). Если промахнулся, граната падает в 1d6 метрах от цели в случайном направлении.

При взрыве все цели в радиусе получают урон. Если цель в укрытии, взрыв сначала разрушает укрытие (сравни урон с ХП укрытия), остаток урона идет по цели.

### Смерть
Когда у тебя **0 ХП**, ты в отключке. В начале своего хода кидай 1d20:
- **10+** — держишься (ничего не меняется)
- **Меньше 10** — провал (нужно 3 провала, чтобы умереть навсегда)

Если союзник потратит действие и кинет [[0.0_Правила#5. НАВЫКИ|Медицину]] (СЛ 12), ты приходишь в себя с 1 ХП.

## 5. НАВЫКИ

Если Мастер просит кинуть навык, кидай `1d20 + нужная характеристика + бонус мастерства`, если этот навык есть в списке твоей роли.

| Навык           | Стат | Когда кидать |
| :-------------- | :--- | :------------------------------- |
| **Атлетика**    | СИЛ  | Лазаешь, прыгаешь, ломаешь двери |
| **Восприятие**  | ИНТ  | Ищешь засаду, замечаешь детали   |
| **Скрытность**  | ЛОВ  | Крадешься, прячешься             |
| **Убеждение**   | ХАР  | Врешь, уговариваешь              |
| **Запугивание** | ХАР  | Пугаешь, давишь                  |
| **Техника**     | ИНТ  | Чинишь, разбираешься в механике  |
| **Медицина**    | ИНТ  | Лечишь, стабилизируешь           |
| **Стрельба**    | ЛОВ  | Атаки дальнего боя               |
| **Бой**         | СИЛ  | Атаки ближнего боя               |

## 6. КИБЕРНЕТИКА

У тебя есть **5 очков** под импланты. Один имплант = одно очко. Если поставить 6-й — киберпсихоз, персонаж уходит к Мастеру.

Некоторые импланты (глаза, руки, ноги) устроены сложнее:
1. Ты ставишь **базу** — это тратит 1 очко.
2. База дает **пустые слоты под опции** (указано в описании).
3. Ты покупаешь **опции** и вставляешь их в эти слоты.
4. База без опций **не дает никаких бонусов**.

Можно потратить **еще 1 очко**, чтобы расширить базу и получить **+3 слота под опции**. Апгрейд делается один раз для одной базы.

Подробнее — в [[0.2_Импланты|Имплантах]].

## 7. ДРОНЫ И УПРАВЛЕНИЕ

Дроны управляются действием. Если у тебя есть [[0.2_Импланты#Личный порт|Личный порт]] и [[0.2_Импланты#Нейролинк|Нейролинк]], ты можешь отдавать команды мыслью (действие не требуется, но не больше одной команды за раунд).

Дроны бывают разные — смотри в [[0.1_Снаряжение#7. ДРОНЫ|Снаряжении]].
